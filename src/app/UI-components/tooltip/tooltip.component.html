Wie wird gespielt

  1. Du wählst zuerst ein Würfel-Preset aus.
  2. Du drückst auf Start
  3. Druch drücken auf einen der Würfel wird eine Seite zufällig gewürfelt
  4. Dann machst du das gleiche mit dem 2. Würfel
  5. Jetzt muss das getan werden, was der Würfel anzeigt
  6. Wenn die Aufgaben erfüllt wurden, drückst du auf den Button "Nächster"
